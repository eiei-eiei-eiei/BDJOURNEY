# คู่มือเริ่มใช้ Claude Code (สำหรับมือใหม่ Windows)

ทำตามลำดับ ทีละขั้น อย่าข้าม ถ้าขั้นไหนพิมพ์แล้วขึ้น error ให้หยุดแล้วถามก่อน

---

## ขั้น 0 — ติดตั้ง Node.js (ทำครั้งเดียว)
Claude Code กับ clasp ต้องใช้ Node.js

1. เปิดเบราว์เซอร์ไป https://nodejs.org
2. โหลดปุ่มฝั่งซ้าย **LTS** (เวอร์ชันแนะนำ)
3. เปิดไฟล์ที่โหลด กด Next ไปเรื่อยๆ จนเสร็จ (ค่า default ได้หมด)

---

## ขั้น 1 — เปิด PowerShell
1. กดปุ่ม **Windows** บนคีย์บอร์ด
2. พิมพ์ `powershell`
3. กด **Enter** → ได้หน้าต่างสีน้ำเงิน/ดำ ให้พิมพ์คำสั่ง

> เคล็ดลับ: คัดลอกคำสั่งจากคู่มือนี้ แล้ว **คลิกขวา** ในหน้าต่าง PowerShell = วาง (Ctrl+V ก็ได้)

---

## ขั้น 2 — เช็คว่า Node.js ติดตั้งสำเร็จ
พิมพ์บรรทัดนี้ กด Enter:
```
node --version
```
ถ้าขึ้นเลขเช่น `v22.x.x` = สำเร็จ ✅
ถ้าขึ้น "ไม่รู้จักคำสั่ง" = ปิด PowerShell เปิดใหม่ แล้วลองอีกครั้ง (ถ้ายังไม่ได้ ติดตั้ง Node ขั้น 0 ใหม่)

---

## ขั้น 3 — ติดตั้ง Claude Code (ทำครั้งเดียว)
```
npm install -g @anthropic-ai/claude-code
```
รอสักครู่ (มีข้อความวิ่งเยอะ ปกติ) จนกลับมาที่บรรทัดพิมพ์ได้

---

## ขั้น 4 — เข้าโฟลเดอร์โปรเจกต์
```
cd "C:\Users\Admin\OneDrive\BDJ\OurDiary-PWA"
```
`cd` = เข้าโฟลเดอร์ สังเกตว่าด้านซ้ายของบรรทัดจะเปลี่ยนเป็น path นี้

---

## ขั้น 5 — เปิด Claude Code
```
claude
```
- ครั้งแรกจะให้ **login** → กด Enter มันจะเปิดเบราว์เซอร์ให้ล็อกอินบัญชี Claude
- เสร็จแล้วกลับมาที่ PowerShell จะเข้าโหมด Claude Code

ลองพิมพ์สั่งเป็นภาษาไทยได้เลย เช่น:
```
อธิบายโครงสร้างโปรเจกต์นี้ให้หน่อย
```

**ออกจาก Claude Code:** พิมพ์ `/exit` หรือกด Ctrl+C สองครั้ง

---

## ขั้น 6 (ทำทีหลัง) — ต่อ clasp เพื่อ sync Code.gs ขึ้น Apps Script
ยังไม่ต้องทำตอนนี้ ค่อยทำเมื่อชินกับ Claude Code แล้ว
```
npm install -g @google/clasp
clasp login
```
แล้วค่อยให้ Claude Code ช่วยตั้งค่าที่เหลือ

---

## สรุปคำสั่งที่ใช้บ่อย
| อยากทำ | พิมพ์ |
|---|---|
| เข้าโฟลเดอร์ | `cd "C:\Users\Admin\OneDrive\BDJ\OurDiary-PWA"` |
| เปิด Claude Code | `claude` |
| ออก | `/exit` |
| ดูว่าอยู่โฟลเดอร์ไหน | `pwd` |
| ดูไฟล์ในโฟลเดอร์ | `ls` |

ครั้งต่อๆ ไปแค่ทำ **ขั้น 1 → 4 → 5** พอ (ขั้น 0,2,3 ติดตั้งครั้งเดียว)
